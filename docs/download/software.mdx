## Ableton Live

Ableton Live是由Ableton公司制作的一款数字音频工作站（DAW），通过Live可以自由的创作音乐，现场演出，即兴演奏。Live与Launchpad做了深度整合

![img](/img/software/live-150x150.png)

Live目前有以下版本：
* Intro 入门版
* Standard 标准版
* Suite 完整版
* Lite 精简版（与设备一起捆绑，不另卖）
* Trial 试用版

:::tip 注意
硬件设备捆绑的Lite激活到Ableton账户后可以享受85折升级到Suite

仅有Suite版本带Max For Live模块，游玩Launchpad工程请务必购买Suite版本

试用版可90天免费试用完整内容，到期后可继续试用但无法保存/另存为，所以不要拿来做工程（

请不要冲动消费，先用试用版体验也是没有问题的-(:з」∠)-
:::
