众所周知，Launchpad的设备端口名称在Windows与macOS上显示的是不一样的。

由于Novation自身驱动的原因，Launchpad在Ableton Live中并不能显示正确的名称，而会使用MIDIIN/MIDIOUT来替代 （我猜可能是Novation的驱动都是通用的，没有单独给每个设备做驱动）

而macOS则会按照官方设定的名称进行显示

那么本文就是列个表格将Windows上的端口转换成正确的名称
:::caution
如果你的设备名前方有如1-、2-等等这样的前缀，请无视。这个是Windows系统自身同设备多驱动的标识，不影响你的选择与使用
:::
