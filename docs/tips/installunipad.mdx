由于提取的Unipad 4.0测试版安装包采用.apks格式，无法直接进行安装，需要使用到SAI安装器

所以本篇会教大家如何使用SAI安装器来安装4.0测试版的Unipad

----

### 前期准备

首先从这里下载SAI安装器：[点此下载](https://dl.deltaasterism.com/storage/sai/SAI_4.5.apk)

然后准备好Unipad 4.0测试版的安装包：[点此下载](/download/phonesoftware#unipad)

将下载的两个安装包都拷贝到手机你能找到的目录，如根目录

### 安装SAI安装器

首先安装SAI安装器，下载好后使用手机点开安装：

<img src="/img/tutorial/sai-1.jpg" alt="sai tutorial img" width="600"/>

安装完后打开软件，如下图：（如果有权限弹窗，请务必允许）

### 安装Unipad 4.0测试版

点击下面的安装APK

选择内置文件选择器

<img src="/img/tutorial/sai-3.jpg" alt="sai tutorial img" width="600"/>

如有权限弹窗，请全部允许

<img src="/img/tutorial/sai-4.jpg" alt="sai tutorial img" width="600"/>

然后找到你刚拷贝的两个文件的地方，选择Unipad的安装包

<img src="/img/tutorial/sai-5.jpg" alt="sai tutorial img" width="600"/>

这里直接点安装

<img src="/img/tutorial/sai-6.jpg" alt="sai tutorial img" width="600"/>

然后跟着步骤安装即可

<img src="/img/tutorial/sai-7.jpg" alt="sai tutorial img" width="600"/>

<img src="/img/tutorial/sai-8.jpg" alt="sai tutorial img" width="600"/>

然后打开Unipad 4.0，会有弹窗说需要读取文件的授权，请点击Confirm，然后给予Unipad访问储存的权限
